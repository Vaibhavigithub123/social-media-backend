# Social Media backend

Backend mini project built using Node.js, Express, MongoDB, and Mongoose.

## Tech Stack
- Node.js + Express
- MongoDB + Mongoose
- JWT Authentication
- bcrypt
- ejs

## Concepts Practiced
This project was built to practice advance concepts of Mongodb and Express.

- MVC folder structure
- REST API design
- Schema design — embedding vs referencing
- Authentication & authorization
- Pagination
- Indexes
- MongoDB aggregation pipeline
- Global error handling
- Environment variables
